// SETUP GUIDE

    
    
      
  1. GitHub Repo banao — New public repository banao.
    2. 
      
  2. 4 files upload karo — index.html, videos.json, uploader.py, aur .github/workflows/upload.yml
    3. 
      
  3. Google Cloud Console — console.cloud.google.com pe jao → New Project → YouTube Data API v3 enable karo → OAuth 2.0 Client ID banao (Desktop app type) → Client ID aur Secret copy karo.
    4. 
      
  4. get_token.py run karo — Apne PC par pip install google-auth-oauthlib karo, phir python get_token.py run karo. Browser mein apne YouTube channel wale Google account se login karo. REFRESH_TOKEN milega.
    5. 
      
  5. GitHub Secrets add karo — Repo Settings → Secrets → Actions → New secret. 4 secrets add karo:
    YOUTUBE_CLIENT_ID
    YOUTUBE_CLIENT_SECRET
    YOUTUBE_REFRESH_TOKEN
    ANTHROPIC_API_KEY (claude.ai account se API key lo)
    6. 
      
  6. GitHub Pages on karo — Settings → Pages → Source: main branch → Save.
    7. 
      
  7. Links add karo! — Site pe links paste karo → Add to Queue → Export JSON → videos.json ko GitHub pe commit karo. Bas! GitHub Actions 9am aur 7pm IST pe automatically upload karega.
